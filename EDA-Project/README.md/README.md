# Exploratory Data Analysis (EDA) & Business Intelligence

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) and Business Intelligence using a cleaned sales dataset. The objective is to analyze customer purchasing behavior, identify sales trends, and generate business insights through data visualization and statistical analysis.

## Dataset Information

The dataset contains information related to customer orders, products, sales, and demographics.

### Features

* Order_ID
* Order_Date
* Customer_ID
* Customer_Name
* Age
* Gender
* City
* Product
* Category
* Quantity
* Unit_Price
* Total_Sales

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* SQL
* Excel / Power BI

## Tasks Performed

### 1. Data Understanding

* Loaded and explored the dataset.
* Checked dataset structure and column information.
* Generated descriptive statistics.

### 2. Exploratory Data Analysis

* Sales Distribution Analysis
* Customer Demographic Analysis
* Revenue Analysis by Category
* Revenue Analysis by City
* Monthly Sales Trend Analysis
* Correlation Analysis

### 3. Data Visualization

Created visualizations including:

* Histogram
* Bar Charts
* Pie Charts
* Line Charts
* Correlation Heatmap

### 4. SQL Business Queries

Performed SQL analysis to answer business questions such as:

* Top revenue-generating cities
* Best-selling products
* Highest revenue categories
* Top customers by sales
* Overall sales performance

## Key Insights

* Identified top-performing products and categories.
* Analyzed customer distribution across cities.
* Evaluated monthly sales trends.
* Studied relationships between quantity sold and total sales.
* Generated actionable business insights using data analysis.
